- **现状**: 流式接口等待完整响应后再分块发送

- **解决方案**: 接入 LLM 的 `astream()` 接口
- **代码位置**: `src/ragent_backend/app.py:418-452`

- **问题**: 历史消息可能加载失败时不报错
- **建议**: 添加加载状态指示和错误提示
